# 🇰🇬 Форсайт-симулятор Кыргызской Республики

Интерактивный макроэкономический симулятор на базе модели Правительства КР / СЕКО.

## Быстрый старт

```bash
pip install -r requirements.txt
streamlit run app.py
```

Откроется браузер: http://localhost:8501

## Возможности

| Вкладка | Содержание |
|---|---|
| 📊 Обзор | KPI-карточки, сводный дашборд, тепловая карта, матрица корреляций |
| 🔴 Реальный сектор | ВВП рост, номинальный ВВП, разрыв выпуска, таблица данных |
| 💰 Фискальный сектор | Баланс бюджета, долг/ВВП, DSA-анализ с зонами риска |
| 🏦 Монетарный & Внешний | Инфляция, курс сома, текущий счёт, резервы, золото→курс |
| ⚡ Стресс-тест | 5 шоковых сценариев, таблица рисков 🟢🟡🔴 |

## Параметры (настраиваются в сайдбаре)

- Шок цены золота: −$600 … +$600/унц.
- Шок роста ВВП: ±6 п.п.
- Фискальный шок: ±4% ВВП
- Инфляция США: 1–8%
- Реформы: 0–1
- Горизонт прогноза: 2025–2035

## Калиброванные уравнения (OLS 2000–2023)

```
Inflation    = 0.054 + 0.548×GDP_growth + 0.12×Depreciation + 0.35×US_CPI
Exchange_rate= 31.85 + 0.0227×Gold_price - 5.0×Fiscal_balance
Fiscal_bal   = -0.044 + 0.107×GDP_growth + 0.00003×ΔGold
Debt/GDP     = prev - 6.026×Fiscal - 0.127×GDP_growth - 0.0003×Gold
GDP_growth   = 0.043 + 0.924×Output_gap + 0.15×Fiscal
```

## Ключевые корреляции (из исторических данных)

- Золото ↔ Курс сом: r = **+0.71**
- Золото ↔ Долг/ВВП: r = **−0.76**
- Фис. баланс ↔ Долг: r = **−0.65**
- Разрыв выпуска ↔ ВВП рост: r = **+0.57**
- ВВП рост ↔ Инфляция: r = **+0.34**
